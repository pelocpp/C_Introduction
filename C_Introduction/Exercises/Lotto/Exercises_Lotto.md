[Zurück](./../Exercises.md)

## Übersicht

Schreiben Sie ein Programm, das eine Lottozahlenziehung simuliert.

Mögliche Ausgaben des Programms könnten sein:


```
Ziehung der Lottozahlen:
========================
23 32 28 8 3 18
```

oder auch

```
Ziehung der Lottozahlen:
1. Kugel: 26
2. Kugel: 23
3. Kugel: 1
4. Kugel: 21
5. Kugel: 12
6. Kugel: 5
```
